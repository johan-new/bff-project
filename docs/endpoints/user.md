 `GET /loggedinuser`
Hämtar information om inloggad användare. Ex

``` 
{
  "games": {
    "1": {
      "venue": "Göteborg",
      "players": "[Johan@mail.com, Erik@mail.com, Simon@mail.com]",
      "id": "1",
      "when": "2020-12-10 09:18:41.368"
    }
  },
  "username": "Johan@mail.com"
}
  ```
